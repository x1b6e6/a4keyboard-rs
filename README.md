# a4keyboard-rs

rewritten [`a4keyboard` (c++)](https://github.com/x1b6e6/a4keyboard) to rust

## Usage

```shell
a4keyboard color ffffff # Set white color to all keys
a4keyboard color f00    # Set red color to all keys
a4keyboard disco        # Enter "disco" mode
```
